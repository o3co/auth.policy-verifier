# @o3co/auth-policy-verifier-standalone

Deployable server template for auth.policy-verifier. This is the composition root — it reads config, loads modules, and starts the Express server. Generated by `create-o3co-policy-verifier`.

## Usage

```sh
pnpm install
OAUTH_JWT_SECRET=your-secret pnpm run start
```

## Configuration

Configuration is read from `config/application.conf` (HOCON format). Individual values can be overridden with environment variables.

| Variable | Default | Description |
|---|---|---|
| `HTTP_HOSTNAME` | `0.0.0.0` | Bind hostname |
| `HTTP_PORT` | `3000` | Bind port |
| `HTTP_PATH_PREFIX` | `""` | URL path prefix |
| `OAUTH_JWT_SECRET` | (required) | HMAC-HS256 JWT signing secret |
| `OAUTH_JWT_VALIDATE` | `true` | Whether to verify JWT signature |

## Default Collectors

The following collectors are registered via `builtinCollectorsModule`:

**Attribute collectors** (extract attributes from the JWT payload):

- `PayloadScopeCollector` — extracts OAuth scopes
- `PayloadSubjectIdCollector` — extracts the subject identifier

**Rule collectors** (resolve authorization rules):

- `ResourceActionScopeRuleCollector` — matches resource/action against scope rules
- `ResourceActionPermissionRuleCollector` — matches resource/action against permission rules

The resource parser in use is `DotNotationResourceParser`.

## Adding Custom Modules

Edit `src/main.mts` and add your module to the `modules` array passed to `createApp`:

```ts
import { myModule } from "./my-module.mts";

const app = await createApp({
  pathResolver: import.meta.resolve,
  config,
  modules: [builtinCollectorsModule, myModule],
});
```

## npm Scripts

| Script | Command | Description |
|---|---|---|
| `build` | `tsc` | Compile TypeScript to `dist/` |
| `start` | `node dist/main.mjs` | Start the compiled server |
| `debug` | `NODE_OPTIONS='--conditions=development' tsx watch src/main.mts` | Start with hot-reload for development |
| `test` | `echo 'no tests configured'` | Placeholder (no tests configured) |

## Docker

Build the image and run the container:

```sh
make build
docker run -e OAUTH_JWT_SECRET=secret auth-policy-verifier
```

For local development with Docker Compose:

```sh
make dev
```
